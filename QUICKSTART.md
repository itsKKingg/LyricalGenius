# Quick Start Guide

Get LyricalGenius running in 5 minutes!

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/lyricalgenius.git
cd lyricalgenius

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## First Video

### 1. Upload Audio
- Drag and drop any audio file (MP3, WAV, M4A, OGG)
- Or click the upload zone to browse your files

### 2. Add Lyrics
- Click the "Lyrics" tab in the right panel
- Click "+ Add Line" to add a new lyric line
- Edit the text and adjust the start/end times
- Add as many lines as you need

**Pro tip:** Start times are in seconds. For example, if your first line starts at 2.5 seconds, enter `2.5`.

### 3. Style Your Video

#### Apply a Preset (Quick)
- Open the left sidebar
- Click any viral preset (e.g., "2025 TikTok Hook")
- Your video instantly updates!

#### Customize Manually
- **Text Tab**: Change font, size, color, effects
- **Background Tab**: Pick gradients or solid colors
- **Effects Tab**: Add animations and visualizers

### 4. Preview
- Click the large play button in the center
- Use the timeline at the bottom to scrub through
- Press Space to play/pause

### 5. Export
- Click the "Export" tab
- Choose your format:
  - **9:16** for TikTok/Reels
  - **1:1** for Instagram Feed
  - **16:9** for YouTube
- Select resolution (1080p recommended)
- Click "Export as MP4"

Done! Your viral lyric video is ready to upload.

## Tips & Tricks

### Keyboard Shortcuts
- `Space` - Play/Pause
- `←` `→` - Skip backward/forward 5 seconds
- Click timeline to jump to any time

### Best Practices
1. **Timing**: Set lyric lines 0.2-0.3s before the actual vocal starts for better sync
2. **Text**: Keep lines short (max 2 lines) for readability on mobile
3. **Colors**: High contrast (white text on dark background) works best
4. **Animations**: Use "Scale" or "Fade" for most songs, "Bounce" for upbeat tracks

### Common Issues

**Q: My lyrics don't sync with the music**
A: Adjust the start/end times in the Lyrics tab. You can type precise decimal values like `2.75`.

**Q: The text is too small on mobile**
A: Use font size 48px or larger for 9:16 videos.

**Q: I can't see the visualizer**
A: Make sure:
- Your audio is playing
- Visualizer style is not set to "None"
- Intensity is above 30%

**Q: Export isn't working**
A: Video export is currently a placeholder. Full implementation coming soon!
